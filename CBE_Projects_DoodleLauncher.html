Doodle Launcher is a shortcut management system for your web browser. Like a glorified bookmark list, Doodle Launcher lets you create clickable buttons that links to your websites and organize them neatly in categories.

Screenshots

Latest in 1.15

  • Launch categories
    • Added a button that lets you choose a category and launch all the shortcuts within it.
    • This button can be accessed by using the Category Navigation sidebar.
  • Import and export shortcuts
    • Added a way to import and export your shortcut list.
    • Exporting your shortcut list converts it into a string of text which can then be copied to your clipboard.
    • Importing your shortcut list involves pasting a valid string. Importing it will automatically place the read shortcuts and links into the proper category you choose.
  • Updated the Loading Screen
  • Semicolons (;) are no longer allowed when creating a new item.
  • Small UI tweaks.

Features

  • Manage shortcuts
    • Create shortcuts that you can link into any website URL, and organize them inside categories.
    • Edit or delete any categories or shortcuts easily.
  • Quick Search
    • No need to open a new tab just to search something. You can open the Quick Search Bar inside the launcher and let the launcher search it for you.
    • Easily change the search engine used by this feature such as Ecosio or Google.
  • Customization
    • Personalize Doodle Launcher by changing the wallpaper, color schemes, and certain behaviors.
    • You can also use the preset dark and light themes.
  • Keyboard shortcuts
    • Make using the launcher even easier with the use of keyboard shortcuts.
    • See all of them in Settings > Information > Keyboard Shortcuts.
  • Easy to Update
    • When a new update is available, you can easily download the latest version package inside the launcher in Settings > Launcher Update.

Requirements and permission requests

  • Web browser
  • Permit Javascript code to run on your browser (this is turned on by default in your browser)

Installation instructions

  1. Click "Download Latest" above to download the latest version of Doodle Launcher.
  2. Unzip the download .zip file.
  3. Click "DL_Main.html" to start the program.
